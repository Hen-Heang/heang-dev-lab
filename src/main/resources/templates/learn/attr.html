2. title 속성 (Tooltip)

<a th:href="@{/product/{id}(id=${productId})}" th:attr="title=${tooltipText}"> 상품 보기 </a>
상품 보기 (hover me!)
EN: Hover your mouse over the link to see the tooltip text from Java.
KO: 링크 위에 마우스를 올리면 Java에서 보낸 툴팁 텍스트가 보입니다.

3. 여러 속성 한번에 (Multiple attributes at once)

<img th:attr="src=${imageUrl}, alt=${productName}" />
EN: Both src and alt are set by one th:attr.
KO: src와 alt 두 속성이 th:attr 하나로 설정됩니다.

정리 — th:attr vs 단축어 (Summary)

Use shortcut / 단축어 사용 Use th:attr / th:attr 사용
th:src="${url}" th:attr="data-id=${id}"
th:href="@{/path}" th:attr="title=${tooltip}"
th:value="${val}" th:attr="data-role=${role}"
