Step 3 — Conditions (th:if / th:unless / th:switch)

1. th:if / th:unless — Basic

You are an admin. Admin menu is visible.

You are a regular user.

Rule: th:if = show when TRUE  |  th:unless = show when FALSE (the else branch)
Try: Change userRole to "USER" in the controller → restart → the second line appears instead.

2. th:switch / th:case — Account status

Account is active.

Account is inactive.

Account is suspended.

Unknown account status.

Rule: Only ONE th:case block is ever shown. th:case="*" is the default (catch-all).
Try: Change loginStatus to "DELETED" in the controller → restart → the default case appears.

3. Number comparison

VIP member! Your points: 0

Regular member.

Try: Change point to 500 in the controller → restart → "Regular member" appears.

Practice A — Show different button based on login status

Welcome! You are logged in.

You need to log in.

Rule: th:if and th:unless on a <div> controls the ENTIRE block inside it.
Try: Change isLoggedIn to false in the controller → restart → Login button appears instead.

Practice B — Nested conditions (cart inside login check)

Shopping Cart

Your cart is empty.

Items in cart: 0

Rule: Inner th:if only runs after the outer th:if passes. Think of it as a gate.
Try: Change cartCount to 3 → restart → "Items in cart: 3" appears. Try: Change isLoggedIn to false → restart → the entire cart section disappears.

Practice C — Grade badge (th:switch)

Your grade: ★ VIP ● GOLD ● SILVER ○ Regular

Try: Change userGrade to "VIP", "SILVER", or "BRONZE" and restart each time. "BRONZE" has no case, so th:case="*" handles it → shows "Regular".

Practice D — AND / OR / NOT conditions

You have 0 notifications.

Premium member benefits applied.

You are not logged in. (using th:if + not)

and = both true  |  or = one true  |  not = reverse  |  th:if="${not x}" = same as th:unless="${x}"
Try: Change hasNotification to false → restart → notification message disappears (AND failed). Try: Change userGrade to "SILVER" → restart → premium message disappears (neither VIP nor GOLD).
