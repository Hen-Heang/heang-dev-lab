Step 1.A — Input Attributes

1. th:value — 입력값 미리 채우기 (Pre-fill Input)

Syntax: <input type="text" th:value="${username}" />

EN: The input shows "heang" because the controller did: model.addAttribute("username", "heang")
KO: 컨트롤러에서 model.addAttribute("username", "heang")을 했기 때문에 "heang"이 표시됩니다.

2. th:placeholder — 힌트 텍스트 (Hint Text)

Syntax: <input type="text" th:placeholder="${hint}" />

EN: Click inside the input — the hint text disappears. That is placeholder behavior.
KO: input 안을 클릭하면 힌트 텍스트가 사라집니다. 이것이 placeholder 동작입니다.

3. th:disabled — 비활성화 (Disable Input)

Syntax: <input type="email" th:value="${email}" th:disabled="${isReadOnly}" />

EN: isReadOnly = true → grayed out, cannot type.
Change to false in controller → restart → input becomes editable.
KO: isReadOnly = true → 회색, 입력 불가.
컨트롤러에서 false로 바꾸고 재시작하면 입력 가능해집니다.
