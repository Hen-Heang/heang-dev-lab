Step 1: 기본 변수 출력 (Basic Variable Display)

1. th:text — 변수 출력

Syntax / 문법
<p th:text="${greeting}">placeholder</p>
Result / 결과

여기에 인사말이 나옵니다

EN: ${greeting} reads the value Java put in Model with model.addAttribute("greeting", "...")
KO: ${greeting}는 Java 컨트롤러에서 model.addAttribute("greeting", "...")로 넣은 값을 읽습니다.

2. 문장 중간에 변수 삽입 (Inline Variable in Sentence)

Syntax / 문법
<p>내 점수는 <span th:text="${score}">0</span> 점입니다.</p>
Result / 결과

내 점수는 0 점입니다.

EN: Wrap only the dynamic part in <span> — the surrounding text stays as-is.
KO: 바뀌는 부분만 <span>으로 감싸면 나머지 문장은 그대로 유지됩니다.

3. 삼항 연산자 (Ternary Operator)

Syntax / 문법
<p th:text="${isAdmin} ? '관리자입니다' : '일반 사용자입니다'"></p>
Result / 결과

역할

EN: isAdmin is true in the controller, so "관리자입니다" is shown.
KO: 컨트롤러에서 isAdmin = true로 보냈으므로 "관리자입니다"가 출력됩니다.

4. th:utext — HTML 태그 포함 출력

Syntax / 문법
<p th:utext="'<strong>굵은 텍스트</strong> 출력'"></p>
Result / 결과

HTML 출력

EN: th:text would display the raw HTML string (escaped). th:utext renders it as real HTML.
KO: th:text는 HTML을 문자열로 보여주지만, th:utext는 실제 HTML로 렌더링합니다.

5. th:class — 조건부 CSS 클래스

Syntax / 문법
<p th:class="${isAdmin} ? 'admin-box' : 'user-box'">...</p>
Result / 결과

역할에 따라 CSS 클래스가 바뀝니다

EN: If isAdmin=true → class becomes admin-box (yellow). If false → user-box (blue).
KO: isAdmin=true면 class가 admin-box(노란색), false면 user-box(파란색)가 됩니다.

6. th:classappend — 클래스 추가 (기존 유지)

Syntax / 문법
<p class="box" th:classappend="${hasError} ? 'error'">...</p>
Result / 결과

기본 클래스는 유지되고 조건부 클래스가 추가됩니다

EN: class="box" always stays. If hasError=true, it becomes class="box error".
KO: class="box"는 항상 유지됩니다. hasError=true면 class="box error"가 됩니다.

7. th:style — 동적 인라인 스타일

Syntax / 문법
<p th:style="${isAdmin} ? 'color: red; font-weight: bold;' : 'color: gray;'">...</p>
Result / 결과

관리자는 빨간색, 일반 사용자는 회색

EN: Prefer th:class + CSS over th:style in real projects. th:style is for quick demos.
KO: 실제 프로젝트에서는 th:class + CSS를 권장합니다. th:style은 간단한 테스트용입니다.

8. 리터럴 치환 (Literal Substitution) — |...|

Syntax / 문법
<div th:id="|user-${userId}|">...</div>
Result / 결과 (inspect element to verify)
이 div의 id는 "user-7" 이 됩니다 (브라우저에서 요소 검사로 확인)
EN: |user-${userId}| → produces user-7. Same as 'user-' + ${userId} but cleaner.
KO: |user-${userId}| → user-7이 됩니다. 'user-' + ${userId}와 같지만 더 깔끔합니다.
